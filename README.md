# Schoolsoft
Schoolsoft website with Laravel + Inertiajs (React)

## 1- Install composer packages
```
composer install
```

## 2- Install node dependencies
```
npm install
```

## 3- Run de app
```
php artisan serve
```

Open another terminal and run
```
npm run dev
```

## Dev features (urls)
- /telescope
- /translations (backend)
